# Evaluation results

Metrics from the held-out **test split** (chronological 15% of dates). Generated by `scripts/evaluate_predictions.py` from `data/processed/final_ensemble_predictions.csv`.

**Preset:** `max_v2` — 20 tickers, next-day horizon, FinBERT embeddings, conditional ensemble (`build_ensemble.py --conditional`), VIX regime features, 0.3% min-move label filter.

**Reproduce:**

```bash
python scripts/evaluate_predictions.py --horizon 1
cat data/processed/evaluation_summary.txt
```

Outputs also written to `evaluation_overall.csv`, `evaluation_by_ticker.csv`, `evaluation_by_confidence.csv`.

## Key findings

### 1. Ensemble meta-learning works

Individual base models are weak (50–54% accuracy), but the HistGradientBoosting combiner reaches **62.2%** test accuracy and **0.661 AUC**. The ensemble down-weights the collapsed LSTM and up-weights news models on headline days.

### 2. News signal is real but asymmetric

- **FinBERT embeddings** — best standalone news model (53.9% acc, 0.603 AUC). Conservative on UP calls (26.8% recall).
- **TF-IDF** — accuracy near random (50.9%) but **AUC 0.640** — ranking signal exists; threshold/calibration yields very few UP predictions (10.4% recall).

### 3. LSTM price model underperforms

Test accuracy **49.7%**, AUC **0.495** — effectively random. Training diagnostics flag probability collapse (predictions cluster near one class). The ensemble still includes LSTM features but learns not to rely on them heavily.

Recent training changes (`max_v2`): weighted BCE, val-AUC early stopping, sigmoid-only calibration when raw variance is low. Collapse persists on test — further work needed (regularization, focal loss, or temporary LSTM exclusion).

### 4. Baselines are hard to beat individually

`previous_day_direction` (50.9%) and `always_up` (51.4%) set a low bar. Beating **55% sustained** on a chronological test split is meaningful for this task.

### 5. Confidence gating helps

High-confidence ensemble calls reach **65.9%** accuracy. Confidence is `|P(UP) − 0.5| × 2` — lean strength, not calibrated P(correct). See [DATA.md § Confidence](DATA.md).

## Limitations

| Limitation | Impact |
|------------|--------|
| Chronological test only | No walk-forward or purged cross-validation yet |
| Finnhub free tier | ~1 year of news history; older dates sparse |
| Single test split | Metrics vary with split boundary and date range |
| LSTM collapse | Price head may add noise until retrained or dropped |
| Next-day horizon | 3-day horizon presets exist but UI defaults to 1-day |
