How to read this
Purple line = closing price. Cyan line = ensemble probability the next session closes UP (1.0 fully bullish, 0.5 coin-flip). Dashed vertical line = the date you're viewing. Stepped is one discrete call per day; Smooth draws a trend line over the same data.

How the ensemble has been doing

rolling 7-day accuracy
Rolling share of the last 7 calls that turned out correct. 0.5 = coin-flip.

Full-backtest metrics

Measured on the held-out test set, all tickers.
What are the rows?
  • ensemble — the stacked meta-model powering this dashboard.
  • lstm_price — prices + indicators only (no news).
  • news_tfidf, news_embeddings — news-only models.
  • always_up — trivial baseline (always UP). Reveals market drift.
  • previous_day_direction — predict yesterday's direction. A high score = the market was trending.
Run scripts/evaluate_predictions.py to populate.

Reliability

Is the confidence number trustworthy? On the diagonal = "80% chance UP" is right ~80% of the time. Below the line = overconfident, above = underconfident.

Confidence spread

Where the probability lands across every day we've scored for this ticker. A few tall spikes = plateaus (calibrator was fit on too few rows).
